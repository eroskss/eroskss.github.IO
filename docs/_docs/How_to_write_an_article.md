##### **Markdown语法教程**  

想要在**`项目介绍`**和**`项目文档`**中进行正文编辑，其实很简单。只是需要一些**markdown**的简单语法就可以了。  
下面就来介绍一下**markdown**语法的具体使用规则。  
分别使用“示例”与“最终效果”展示。
  
| markdown语法展示 |
| :------: |
| 示例 |
| 最终效果 |  
  
  
#### 标题  

```markdown
# 这是H1标题
```

# 这是H1标题
  
```markdown
## 这是H2标题
```

## 这是H2标题  
  
```markdown
### 这是H3标题
```

### 这是H3标题  
    
```markdown
#### 这是H4标题
```

#### 这是H4标题  
  
```markdown
##### 这是H5标题
```

##### 这是H5标题  
  
```markdown
###### 这是H6标题
```

###### 这是H6标题  
  
  
 Markdown的标题形式是以“# + 标题内容”实现的。其中#号的数量越多它的字号以及标题层级就会越小。  
 特别要注意的是只有H1————H4的标题才会被右侧的导航自动形成目录【如：#、##、###、####】。
{:.info}
  
#### 区块引用  
  
文章中可以引用别人的话，而引用的方法也可以通过特殊的语法实现，在markdown中是“> + 内容”的形式。  
Markdown 标记区块引用的方法是在行的最前面加>  

```markdown
>这是引用的内容
>>**引用可以和其他语法叠加比如加粗**
>>>>>>>>>>这是引用的内容
```

>这是引用的内容
>>**引用可以和其他语法叠加比如加粗**
>>>>>>>>>>这是引用的内容
  
引用可以多次嵌套，只要在>的基础上再加一个>就可以了，但是务必要在>后加一位空格，不然无法识别。
{:.info}
  
#### 加粗  
```markdown
**我是加粗了的字体**
```
**我是加粗了的字体**  

#### 斜体  
```markdown
*我是倾斜了的字体*
```
*我是倾斜了的字体*  
  
#### 斜体加粗  
```markdown
***我是倾斜加粗了的字体***
```
***我是倾斜加粗了的字体***  
  
#### 删除线  
```markdown
~~我是删除线字体~~
```
~~我是删除线字体~~  
  
#### 分割线  
三个或三个以上的这几个符号都可以
{:.info}
  
```markdown
---
----
***
*****
```
---
----
***
*****  

#### 超链接
语法：  
```markdown
[超链接名](超链接地址 "超链接title")
title可加可不加
```
  
 示例：  
```markdown
[百度](http://baidu.com)
```
[百度](http://baidu.com)  
  
#### 无序列表  
无序列表用 - + * 任何一种都可以  
```markdown
- 列表内容
+ 列表内容
* 列表内容
```  
注意：- + * 跟内容之间都要有一个空格
{:.info}

- 列表内容
+ 列表内容
* 列表内容  
  
#### 有序列表  
语法是数字加点  
```markdown
1. 列表内容
2. 列表内容
3. 列表内容
```  
注意：序号跟内容之间要有空格
{:.info}
1. 列表内容
2. 列表内容
3. 列表内容  

#### 列表嵌套  
```markdown
- 列表内容
	- 列表内容
		- 列表内容
	- 列表内容
- 列表内容
- 列表内容
```  
列表嵌套需要用到缩进，建议使用**Tab**键进行缩进，空格也行，注意“-”后一定要有一位空格。
{:.info}
- 列表内容
	- 列表内容
		- 列表内容
	- 列表内容
- 列表内容
- 列表内容  
  
#### 表格
```markdown
| 表头    |   表头   |     表头 |
|:------|:------:|-------:|
| 内容靠左边 | 这是内容居中 | 这是内容靠右 |
| 内容    |   内容   |     内容 |
```  
  
| 表头    |   表头   |     表头 |
|:------|:------:|-------:|
| 内容靠左边 | 这是内容居中 | 这是内容靠右 |
| 内容    |   内容   |     内容 |

  
第二行分割表头和内容。  
-有一个就行，为了对齐，多加了几个  
文字默认居左  
-两边加：表示文字居中  
-右边加：表示文字居右  
注：原生的语法两边都要用 | 包起来。此处省略
{:.info}


  	代码...
